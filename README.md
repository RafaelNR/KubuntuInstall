
## INSTALL - Executar o Script

mkdir /etc/Suporte

cd /etc/Suporte 

sudo wget https://raw.githubusercontent.com/RafaelNR/KubuntuInstall/master/Insall-Kubuntu.sh 

chmod +x Install-Kubuntu.sh 

./Install-kubuntu.sh
